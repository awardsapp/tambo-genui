This template app shows how to setup the fundamental parts of an AI application using genui:

**Component registration**

See in [src/lib/genui.ts](https://github.com/genui-ai/genui-template/blob/main/src/lib/genui.ts) how a graph component is registered with genui.

**Tool Registration**

See in [src/lib/genui.ts](https://github.com/genui-ai/genui-template/blob/main/src/lib/genui.ts) how population data tools are registered with genui.

**UI for sending messages to genui and showing responses**

The components used within [src/components/genui/message-thread-full.tsx](https://github.com/genui-ai/genui-template/blob/main/src/components/genui/message-thread-full.tsx) use hooks from genui's react SDK to send messages and show the thread history.

**Wrap the app with the `GenuiProvider`**

In [src/app/chat/page.tsx](https://github.com/genui-ai/genui-template/blob/main/src/app/chat/page.tsx) we wrap the page with the `GenuiProvider` to enable the usage of genui's react SDK within the message sending and thread UI components.
