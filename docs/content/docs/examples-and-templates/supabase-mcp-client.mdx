Add MCP server tools and UI tools to a React app using genui to build an AI app with minimal custom code.

Use this as a starting point to build apps to interact with any MCP server.

[Github](https://github.com/genui-ai/supabase-mcp-client/tree/main?tab=readme-ov-file#supabase-mcp-client-react-app)

This application makes use of Genui's MCP integration to easily add the tools defined by the official [Supabase MCP server](https://github.com/supabase-community/supabase-mcp).

Custom react components to show interactive visualizations of the responses from the Supabase tools are registered with genui in [src/lib/genui.ts](https://github.com/genui-ai/supabase-mcp-client/blob/main/src/lib/genui.ts)
