## Try our CLI command

The quickest way to add Genui components to your project is with our CLI:

```bash
npx genui full-send
```

This will install a set of common components and hook them up to Genui in your project.

For more customization options and advanced component usage, refer to the component documentation at [ui.genui.co](https://ui.genui.co).
