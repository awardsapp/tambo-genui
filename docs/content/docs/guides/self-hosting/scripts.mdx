## Shell scripts

| Script                             | Description                         |
| ---------------------------------- | ----------------------------------- |
| `./scripts/cloud/genui-setup.sh`   | First-time environment setup        |
| `./scripts/cloud/genui-start.sh`   | Start all services                  |
| `./scripts/cloud/genui-stop.sh`    | Stop all services                   |
| `./scripts/cloud/genui-logs.sh`    | View logs (all or specific service) |
| `./scripts/cloud/genui-build.sh`   | Build Docker images                 |
| `./scripts/cloud/init-database.sh` | Run database migrations             |
| `./scripts/cloud/genui-psql.sh`    | PostgreSQL CLI access               |

### genui-logs.sh

Accepts an optional service name argument:

```bash
./scripts/cloud/genui-logs.sh           # All services
./scripts/cloud/genui-logs.sh postgres  # PostgreSQL only
./scripts/cloud/genui-logs.sh api       # API only
./scripts/cloud/genui-logs.sh web       # Web only
```

## npm convenience aliases

These are shorthand for the shell scripts above. They require **bash**, so Windows users should run them in **WSL** or **Git Bash**.

| Command               | Equivalent script                |
| --------------------- | -------------------------------- |
| `npm run genui:setup` | `./scripts/cloud/genui-setup.sh` |
| `npm run genui:start` | `./scripts/cloud/genui-start.sh` |
| `npm run genui:stop`  | `./scripts/cloud/genui-stop.sh`  |
