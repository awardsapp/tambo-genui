The `@workspace/react` package is Genui's React SDK, featuring a streaming-first architecture, explicit thread management, and React Query integration.

## Installation

```bash
npm install @workspace/react
```

## Quick Start

```tsx
import { GenuiProvider, useGenui, useGenuiThreadInput } from "@workspace/react";
```

## Overview

The React SDK is organized around a simplified provider hierarchy with React Query for data fetching:

```tsx
import { GenuiProvider } from "@workspace/react";

function App() {
  return (
    <GenuiProvider
      apiKey={process.env.NEXT_PUBLIC_GENUI_API_KEY}
      userKey={userId}
      components={
        [
          /* your components */
        ]
      }
      tools={
        [
          /* your tools */
        ]
      }
    >
      <YourApp />
    </GenuiProvider>
  );
}
```

Inside the provider, use hooks to access Genui functionality:

```tsx
import { useGenui, useGenuiThreadInput } from "@workspace/react";

function Chat() {
  const { messages, isStreaming, startNewThread } = useGenui();
  const { value, setValue, submit } = useGenuiThreadInput();

  return (
    <div>
      {messages.map((msg) => (
        <Message key={msg.id} message={msg} />
      ))}
      <input
        value={value}
        onChange={(e) => setValue(e.target.value)}
        onKeyDown={(e) => e.key === "Enter" && submit()}
      />
    </div>
  );
}
```

## Key Features

| Feature          | Implementation                                                                         |
| ---------------- | -------------------------------------------------------------------------------------- |
| Import path      | `@workspace/react`                                                                      |
| Main hook        | `useGenui()`                                                                           |
| User scoping     | `userKey` prop                                                                         |
| Thread input     | `useGenuiThreadInput()`                                                                |
| Message format   | Content blocks (text, component, tool_use, etc.)                                       |
| Component state  | `useGenuiComponentState()` (bidirectional sync)                                        |
| Streaming status | `streamingState.status` (typed as [`RunStatus`](/reference/react-sdk/types#runstatus)) |
| Data fetching    | React Query hooks                                                                      |
