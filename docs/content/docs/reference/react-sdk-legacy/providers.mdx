Provider components configure Genui functionality and make hooks available throughout your component tree.

## GenuiProvider

The main provider that wraps your application and provides access to the full Genui API. This is the primary way to integrate Genui into your React app.

```tsx
import { GenuiProvider } from "@workspace/react";

function App() {
  return (
    <GenuiProvider
      apiKey={process.env.GENUI_API_KEY}
      components={myComponents}
      tools={myTools}
    >
      <YourApp />
    </GenuiProvider>
  );
}
```

### Props

| Prop                        | Type                          | Required | Description                                      |
| --------------------------- | ----------------------------- | -------- | ------------------------------------------------ |
| `apiKey`                    | `string`                      | Yes\*    | Your Genui API key                               |
| `userToken`                 | `string`                      | No       | OAuth token for user authentication              |
| `genuiUrl`                  | `string`                      | No       | Custom Genui API URL                             |
| `environment`               | `string`                      | No       | Environment name                                 |
| `components`                | `GenuiComponent[]`            | No       | Components to register                           |
| `tools`                     | `GenuiTool[]`                 | No       | Tools to register                                |
| `mcpServers`                | `McpServerInfo[]`             | No       | MCP servers to connect                           |
| `contextHelpers`            | `ContextHelpers`              | No       | Context helper functions                         |
| `contextKey`                | `string`                      | No       | Key for thread scoping                           |
| `streaming`                 | `boolean`                     | No       | Enable streaming (default: `true`)               |
| `autoGenerateThreadName`    | `boolean`                     | No       | Auto-generate thread names (default: `true`)     |
| `autoGenerateNameThreshold` | `number`                      | No       | Message count for name generation (default: `3`) |
| `initialMessages`           | `InitialGenuiThreadMessage[]` | No       | Initial messages for new threads                 |
| `onCallUnregisteredTool`    | `function`                    | No       | Callback for unregistered tool calls             |
| `resources`                 | `ListResourceItem[]`          | No       | Static resources for MCP                         |
| `listResources`             | `function`                    | No       | Dynamic resource listing                         |
| `getResource`               | `function`                    | No       | Resource content resolver                        |

\*Either `apiKey` or `userToken` is required for authentication.

### Example with All Options

```tsx
import {
  GenuiProvider,
  currentPageContextHelper,
  type GenuiComponent,
  type GenuiTool,
} from "@workspace/react";

const components: GenuiComponent[] = [
  {
    name: "WeatherCard",
    description: "Displays weather information",
    component: WeatherCard,
    propsSchema: z.object({
      city: z.string(),
      temperature: z.number(),
    }),
  },
];

const tools: GenuiTool[] = [
  {
    name: "get_weather",
    description: "Fetch weather for a city",
    tool: async ({ city }) => fetchWeather(city),
    inputSchema: z.object({ city: z.string() }),
    outputSchema: z.object({ temperature: z.number() }),
  },
];

<GenuiProvider
  apiKey={process.env.GENUI_API_KEY}
  components={components}
  tools={tools}
  contextHelpers={{
    currentPage: currentPageContextHelper,
  }}
  streaming={true}
  autoGenerateThreadName={true}
  autoGenerateNameThreshold={3}
>
  <App />
</GenuiProvider>;
```

### Provider Hierarchy

`GenuiProvider` internally nests several sub-providers in this order:

1. `GenuiClientProvider` - API client and authentication
2. `GenuiRegistryProvider` - Component and tool registration
3. `GenuiContextHelpersProvider` - Context helper management
4. `GenuiThreadProvider` - Thread state and messaging
5. `GenuiMcpTokenProvider` - MCP token management
6. `GenuiMcpProvider` - MCP server connections
7. `GenuiContextAttachmentProvider` - Context attachments
8. `GenuiComponentProvider` - Component lifecycle
9. `GenuiInteractableProvider` - Interactable components
10. `GenuiThreadInputProvider` - Input handling

For advanced use cases, you can use individual providers directly instead of `GenuiProvider`.

## GenuiStubProvider

A stub provider for testing and development that doesn't require API connectivity.

```tsx
import { GenuiStubProvider } from "@workspace/react";

function TestApp() {
  return (
    <GenuiStubProvider
      stubResponses={[
        {
          component: {
            componentName: "WeatherCard",
            props: { city: "Seattle", temperature: 65 },
          },
        },
      ]}
    >
      <YourComponent />
    </GenuiStubProvider>
  );
}
```

### Props

| Prop             | Type                   | Description              |
| ---------------- | ---------------------- | ------------------------ |
| `stubResponses`  | `GenuiThreadMessage[]` | Pre-configured responses |
| `components`     | `GenuiComponent[]`     | Components to register   |
| `tools`          | `GenuiTool[]`          | Tools to register        |
| `contextHelpers` | `ContextHelpers`       | Context helpers          |

## GenuiMcpProvider

Provider for Model Context Protocol (MCP) server connections. Required when using MCP features.

```tsx
import { GenuiMcpProvider } from "@workspace/react/mcp";

// Inside GenuiProvider
<GenuiMcpProvider
  handlers={{
    elicitation: async (request, extra, serverInfo) => {
      // Handle elicitation requests
      return { action: "accept", content: {} };
    },
  }}
  contextKey="my-context"
>
  <App />
</GenuiMcpProvider>;
```

### Props

| Prop         | Type                  | Description                           |
| ------------ | --------------------- | ------------------------------------- |
| `handlers`   | `ProviderMCPHandlers` | Optional handlers for all MCP servers |
| `contextKey` | `string`              | Context key for threadless MCP tokens |
| `children`   | `ReactNode`           | Child components                      |

**Note**: MCP servers are configured on `GenuiProvider` via the `mcpServers` prop. `GenuiMcpProvider` manages the connections and provides hooks to interact with them.

### MCP Server Configuration

Configure MCP servers on `GenuiProvider`:

```tsx
<GenuiProvider
  apiKey={process.env.GENUI_API_KEY}
  mcpServers={[
    {
      name: "linear",
      url: "https://linear-mcp.example.com",
      transport: "http",
      serverKey: "linear",
    },
    {
      name: "github",
      url: "https://github-mcp.example.com",
      transport: "http",
      serverKey: "github",
      customHeaders: {
        Authorization: `Bearer ${githubToken}`,
      },
    },
  ]}
>
  <GenuiMcpProvider>
    <App />
  </GenuiMcpProvider>
</GenuiProvider>
```

See the [MCP reference](/reference/react-sdk/mcp) for hooks and types related to MCP functionality.

## Individual Providers

For advanced use cases, you can use individual providers directly:

### GenuiClientProvider

Provides the API client and authentication context.

```tsx
import { GenuiClientProvider, useGenuiClient } from "@workspace/react";

<GenuiClientProvider
  apiKey={process.env.GENUI_API_KEY}
  genuiUrl="https://api.genui.co"
>
  <App />
</GenuiClientProvider>;
```

### GenuiRegistryProvider

Manages component and tool registration.

```tsx
import { GenuiRegistryProvider, useGenuiRegistry } from "@workspace/react";

<GenuiRegistryProvider
  components={components}
  tools={tools}
  mcpServers={mcpServers}
>
  <App />
</GenuiRegistryProvider>;
```

### GenuiThreadProvider

Manages thread state and message sending.

```tsx
import { GenuiThreadProvider, useGenuiThread } from "@workspace/react";

<GenuiThreadProvider
  contextKey="chat"
  streaming={true}
  autoGenerateThreadName={true}
>
  <App />
</GenuiThreadProvider>;
```

### GenuiContextHelpersProvider

Manages context helpers that provide additional information to the AI.

```tsx
import {
  GenuiContextHelpersProvider,
  useGenuiContextHelpers,
} from "@workspace/react";

<GenuiContextHelpersProvider
  contextHelpers={{
    currentPage: () => ({ url: window.location.href }),
  }}
>
  <App />
</GenuiContextHelpersProvider>;
```

### GenuiInteractableProvider

Manages interactable component registration and state.

```tsx
import {
  GenuiInteractableProvider,
  useGenuiInteractable,
} from "@workspace/react";

<GenuiInteractableProvider>
  <App />
</GenuiInteractableProvider>;
```
