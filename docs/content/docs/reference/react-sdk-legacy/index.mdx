The `@workspace/react` package is Genui's official React SDK for building AI-powered generative UI applications. This reference documents all public APIs including hooks, types, utilities, and provider components.

## Installation

```bash
npm install @workspace/react
```

## Overview

The SDK is organized around a provider hierarchy that manages AI state and configuration:

```tsx
import { GenuiProvider } from "@workspace/react";

function App() {
  return (
    <GenuiProvider
      apiKey={process.env.GENUI_API_KEY}
      components={
        [
          /* your components */
        ]
      }
      tools={
        [
          /* your tools */
        ]
      }
    >
      <YourApp />
    </GenuiProvider>
  );
}
```

Inside the provider, use hooks to access Genui functionality:

```tsx
import {
  useGenui,
  useGenuiThread,
  useGenuiStreamStatus,
} from "@workspace/react";

function Chat() {
  const { sendThreadMessage, thread } = useGenuiThread();
  const { streamStatus } = useGenuiStreamStatus();
  // Build your UI
}
```

## MCP Support

For Model Context Protocol integrations, import from the `/mcp` subpath:

```tsx
import { GenuiMcpProvider, useGenuiMcpServers } from "@workspace/react/mcp";
```

MCP requires additional peer dependencies. See the [MCP reference](/reference/react-sdk/mcp) for setup instructions.
